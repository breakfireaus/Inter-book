# Inter-Book

## Description

A place where people can book services and offer services. this was developed so people could find an easy way to connect to a service they need.


## Technologies used

- bcrypt
- sequelize
- cookies
- dotenv
- express
- express-handlebars
- express-session
- handlebars
- ics
- mysql2
- session
- router
